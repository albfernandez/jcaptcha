The binary distribution of JCaptcha contains the following artefacts :

 - docs : Java apidocs
 - backgrounds (fisheye and gimpy) : in order to generate corresponding captcha
 - extensions : optional jars to include in your classpath
 - integrations : optional jars to include in your classpath to use as integration
 - sources: source code of the library
 - jcaptcha-api.jar : contains core classes including JHLabs imaging api
 - jcaptcha.jar : contains core classes including JHLabs imaging api
 - LICENSE.txt
